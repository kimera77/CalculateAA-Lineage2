# **App Name**: AdenaCalc L2

## Core Features:

- Stone Input: Allow users to input the number of each type of Steal Stone (color-coded: Red, Blue, Green) they possess.
- Calculation Engine: Calculates the total Ancient Adena (AA) yield based on the number of each type of Steal Stone using predefined conversion rates from Lineage 2 (Gracia Final / Interlude).
- Output Display: Displays the total calculated AA in a clear, readable format.
- L2 Image integration: AI powered feature, searches the web to find random and appealing screenshots and photos related to the Interlude/Gracia Final versions of Lineage 2. Generative AI tool classifies images to maximize relevance, choosing from several images for presentation to the user.

## Style Guidelines:

- Primary color: Dark gray (#333333) to provide a sense of classic Lineage 2 mood.
- Background color: Very dark gray (#222222), almost black, a desaturated version of the primary color.
- Accent color: Pale orange/gold (#D4A773), reminiscent of the UI and treasure in Lineage 2, to highlight important elements and buttons.
- Body and headline font: 'Inter', sans-serif.
- Icons that visually represent the different steal stone colors (red, blue, green) and ancient adena.
- Simple, intuitive layout with clear sections for input, calculation, and output, keeping user experience at the forefront.
- Subtle animations when calculating and displaying the AA result to enhance user engagement.
